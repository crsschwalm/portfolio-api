# portfolio-api

AWS + Express api for my portfolio site

### Deploy by

1. Install `npm install -g serverless`
2. Set api Keys `serverless config credentials --provider aws --key xxxxxxxxxxxxxx --secret xxxxxxxxxxxxxx`
3. Deploy `serverless deploy`
